Getting Started

What is Jedison?

Jedison is both a JSON schema validator and a JSON editor.

Features

  • Dependency free
  • JSON Schema Validation
  • JSON Editing by generating forms from JSON schemas
  • Dereferences JSON Schema '$ref' pointers
  • CSS libraries Integration:
    • Bootstrap 3
    • Bootstrap 4
    • Bootstrap 5
  • Icon libraries Integration:
    • Glyphicons
    • Bootstrap icons
    • FontAwesome 3
    • FontAwesome 4
    • FontAwesome 5
    • FontAwesome 6
  • Markdown Support with marked.js (if installed)
  • Secure HTML Sanitization with DomPurify (if installed)
  • Plugin Editors:
    • Quill - powerful rich text editor
    • Flatpickr - lightweight and powerful datetime picker
    • Awesomplete - Ultra lightweight, customizable, simple autocomplete widget with zero dependencies
    • Jodit - WYSIWYG Editor
    • Raty - star Rating Plugin
    • IMask - vanilla javascript input mask
    • EmojiButton - Vanilla JavaScript Emoji Picker

Installing

Using npm:

npm install jedison

Using yarn:

yarn add jedison

Using jsDelivr CDN:

<script src="https://cdn.jsdelivr.net/npm/jedison@0.1.1/dist/umd/jedison.umd.js"></script>

Using unpkg CDN:

<script src="https://app.unpkg.com/jedison@0.1.1/files/dist/umd/jedison.umd.js"></script>
