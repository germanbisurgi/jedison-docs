IMask Number Editor

Renders a masked text input using IMask.js.

Activation Conditions

  • "type": "number"
  • "x-format": "imask"
  • IMask.js must be available (window.IMask)

Plugin Options

Options can be passed to the plugin via "x-imask" keyword
