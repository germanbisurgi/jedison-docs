Themes

Jedison supports multiple CSS frameworks through its theme system, allowing you to maintain consistent styling with your application's design system.

Supported Themes

Default Theme

Basic unstyled theme that provides minimal structure

new Jedison.Theme()

Bootstrap 3

Full support for Bootstrap 3 form styling

new Jedison.ThemeBootstrap3()

Bootstrap 4

Full support for Bootstrap 4 form styling

new Jedison.ThemeBootstrap4()

Bootstrap 5

Full support for Bootstrap 5 form styling

new Jedison.ThemeBootstrap5()

Icon Library Integration

Jedison supports multiple icon libraries which can be specified using the iconLib option:

  • 'glyphicons' - Bootstrap 3's Glyphicons
  • 'bootstrap-icons' - Bootstrap Icons
  • 'fontawesome3' - Font Awesome 3
  • 'fontawesome4' - Font Awesome 4
  • 'fontawesome5' - Font Awesome 5
  • 'fontawesome6' - Font Awesome 6
